# Number Guessing Game 

A simple Java number guessing game where the player tries to guess a random number between 1-100.

# How to Play
1. Computer picks a random number between 1-100
2. You have 7 attempts to guess it
3. Computer tells you if your guess is too high or too low
4. Try to guess the number in fewest attempts!

# How to Run
```bash
javac NumberGame.java
java NumberGame
